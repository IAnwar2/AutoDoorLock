# AutoDoorLock

## Overview
AutoDoorLock is a project designed to automatically lock doors using a combination of sensors and actuators. This system enhances security by ensuring that doors are always locked when not in use.

## Features
- Automatic door locking based on image detection
- Manual override for emergency situations
- Register and delete faces through web app

## Installation
1. Clone the repository:
    ```
    git clone https://github.com/yourusername/AutoDoorLock.git
    ```
2. Navigate to the project directory:
    ```
    cd AutoDoorLock
    ```
3. Follow the setup instructions in the `INSTALL.md` file.

## Usage
1. Ensure all hardware components are properly connected.
2. Run the main application:
    ```
    python main.py
    ```
3. The system will start monitoring and locking doors automatically.
